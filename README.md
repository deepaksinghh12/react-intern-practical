# React Internship Practical (PR: REACTIIP01002)

Simple React app to demonstrate registration, login, and profile management using Bootstrap and localStorage.

## Features
✅ Register new users  
✅ Login & logout  
✅ Edit profile details  
✅ Bootstrap UI  
✅ Error handling & form validation  

## How to run
1. Clone this repo  
   git clone <repo-url>
   cd react-intern-practical
   npm install
   npm start
